# React Todo App..

A modern Todo List application built using **React**.



##  Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as done
- Mark all tasks as done
- Dark mode support
- Smooth UI animations



## Tech Stack
- React (Hooks)
- JavaScript
- CSS


## Preview
Light Mode & Dark Mode supported.

## Future Improvements
1. Save tasks using localStorage
2. Responsive design
